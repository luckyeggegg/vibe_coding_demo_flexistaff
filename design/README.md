## 🎨 设计风格概览

### 核心特点

**简约 · 优雅 · 专业 · 高效**

1. **简约风格**
   - 功能优先，不做过度装饰
   - 清晰的信息层级
   - 直观的操作流程

2. **莫兰迪色系**
   - 低饱和度配色
   - 灰调柔和
   - 适合长时间使用
   - 减少视觉疲劳

### 主色调

| 颜色 | HEX | 用途 |
|------|-----|------|
| 莫兰迪蓝 | `#7C9CB5` | 主要操作 |
| 莫兰迪绿 | `#8FAF9A` | 成功状态 |
| 莫兰迪橙 | `#D4A574` | 警告提示 |
| 莫兰迪红 | `#C29A9A` | 错误提示 |
